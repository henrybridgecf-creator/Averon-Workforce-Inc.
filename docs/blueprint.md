# **App Name**: Investec Private Mock Bank

## Core Features:

- User Authentication: Secure user authentication system with Admin and User login roles.
- User Dashboard: Client app dashboard with account balance, transaction summary, and quick transfer options.
- Admin Dashboard: Admin dashboard with user management, transaction monitoring, and announcement features.
- Mock Transactions: Simulate transactions for demonstration purposes, mimicking real banking interactions. Approving or declining simulated transactions is possible in the Admin dashboard.
- Data Storage: Store mock data and user information, leveraging Firestore for scalable and efficient data management.
- Password Reset: Password reset and recovery tool for users.
- Account Type Display: Checking, Savings and Investments types of accounts displayed and visible.

## Style Guidelines:

- Primary color: Desaturated dark gray (#333333) to give a premium, minimalist base, emulating the bank's elegance.
- Background color: Nearly black (#0A0A0A), for a high-end, dark theme experience.
- Accent color: Light gray (#D3D3D3), providing subtle contrast and a touch of modernity to buttons and key UI elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for headlines or body text.
- Use minimalist icons with a consistent stroke width to maintain a clean and professional look.
- Employ a grid-based layout system with rounded corners and subtle shadows to enhance visual appeal and maintain consistency across the app.
- Implement subtle, premium-feel transitions and animations to provide a smooth and engaging user experience.
